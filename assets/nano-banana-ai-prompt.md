**Cover dimensions:** 8.5" x 11" (letter) or 612 x 792px at 72dpi

**For the fan arrangement:**
1. Create all 13 covers using the template above (just swap the vertical name)
2. Arrange 3 covers in front at slight angles (-5°, 0°, +5°)
3. Stack remaining covers behind at increasing angles
4. Add drop shadow (0px 4px 20px rgba(0,0,0,0.12)) to each
5. Export at 2x for retina
